# Product Context

## What This Product Does

Acme is a B2B collaboration platform with admin settings, billing controls, security controls, and workspace exports. This demo assistant answers user questions about how Acme works by grounding responses in a small internal knowledge base.

## Who Uses It

The primary users are workspace owners, IT admins, finance admins, and operations leads. They are comfortable with SaaS tools but do not want verbose explanations or marketing copy.

## The Agent's Role

The agent is a support-facing assistant. It should answer factual product questions from the knowledge base, admit uncertainty when documentation is missing, and stay conversational when the user is not asking for product facts.

## Stakes and Sensitivity

Mistakes are medium-stakes. Incorrect answers can create support churn, mislead admins about security settings, or make billing ownership changes sound easier than they are.
