# Code of Conduct

## Our Pledge

We want Parity to be a respectful, technically serious, and collaborative open source project. Contributors and maintainers should make participation harassment-free for everyone, regardless of age, body size, disability, ethnicity, sex characteristics, gender identity or expression, level of experience, education, socio-economic status, nationality, personal appearance, race, religion, or sexual identity and orientation.

## Our Standards

Examples of behavior that contributes to a positive environment:

- being respectful and precise in technical discussion
- assuming good intent while still challenging weak reasoning
- accepting feedback gracefully
- focusing on what will improve the project for users and contributors

Examples of unacceptable behavior:

- harassment, abuse, or discriminatory language
- personal attacks or sustained antagonism
- publishing private information without permission
- conduct that would be inappropriate in a professional setting

## Enforcement Responsibilities

Project maintainers are responsible for clarifying and enforcing this code of conduct and may remove, edit, or reject comments, commits, code, issues, and other contributions that are not aligned with it.

## Scope

This code of conduct applies within project spaces and in public spaces when someone is representing the project.

## Reporting

Report unacceptable behavior by opening a private security or maintainer contact route if one is available, or by contacting the repository owner through GitHub.

## Enforcement

Maintainers may take any action they deem appropriate in response to unacceptable behavior, including warnings, temporary bans, or permanent bans.
